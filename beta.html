Quick start

If you don't need a lot of detail or context, here's the tl;dr version (scroll down for the same steps in more detail):

Install

  1. Check that your system meets all the prerequisites (Mac only; Sketch 43; Developers also need node, npm, git, and our dev tools installed).
  2. Download the latest release (Contact us for a link,) extract it, and move Haiku.app to your Applications folder.
  3. Install our dev tools by running this command: curl http://code.haiku.ai/scripts/cli/installer.js | node

Design

  1. Open Haiku.app, wait for it to launch, and sign in with your username and password. (Don't have a login? Contact us.)
  2. Create and select a project. Click the "Open in Editor" button.
  3. Animate! (See our demo video for ideas.)
  4. Click the floppy disk icon on the top-right to save changes (This is temporary until realtime sync is complete.)
  5. Go back to the project dashboard. Copy the installation code snippet for your animation. ( haiku install ... )

Code

  1. Install your animation in a React codebase: haiku install ...
  2. Import the animation into your code: import HaikuAnimation from '@haiku/{project}/react';
  3. Embed the animation in your render method: <HaikuAnimation />
  4. Refresh and watch your animation come to life.

Detailed Instructions

Prerequisites

Today, Haiku can only run on a system that meets a handful of prerequisites. Before continuing, please make sure your system has all of the following. (Big improvements on this are coming soon.)

  • Mac OS 10.9 (Mavericks) or greater
  • Sketch v43
  • Developers: Xcode
  • Developers: Xcode command line tools installed ( xcode-select --install )
  • Developers: git 2.7+
  • Developers: node 6.6+
  • Developers: npm 3.10+

Installing Haiku

Assuming your system checks out, you're ready to install Haiku. We recommend you install both our desktop app and our developer tools so you can experience Haiku as both a designer and an engineer.

  1. Desktop app: Download Haiku. (Contact us for a link.) Unzip the archive, open the extracted folder, and move Haiku.app into your Applications folder.
  2. Developer tools: (You must completely install the desktop app before doing this.) Launch the Terminal application on your Mac, and when you see a command prompt, enter the following snippet: curl http://code.haiku.ai/scripts/cli/installer.js | node

Creating an Animation

With Haiku installed, we're now ready to create an animation. Go ahead and launch the Haiku desktop app. Once it has loaded...

  1. Sign in. (Don't have a login? Contact us.)
  2. Select (or create) a project. Once sign-in has succeeded, you should see a listing of projects including a couple of samples. We suggest you start by editing one of these, but if you prefer a blank slate, you can also create a new project here.
  3. Open the project: Select a project and click the "Open In Editor" button. Wait for the project to load. (This can take several seconds while we set up your project on the cloud for the first time.)
  4. Animate! Compose, move, transform, and tween. For an overview of things you can currently do in Haiku, see our demo video. (Note: We still have a lot of kinks to iron out in our animation editor — UI streamlining, performance improvements, and bug fixes — but we think you'll at least get a glimmer of where we'll be in a few months' time.)
  5. Save. Once you're finished, find the floppy disk icon at top-right and click it to save your changes. You should see a green checkmark if the save went well.
  6. Get the code snippet: Find the arrow icon at top-left and click it to navigate back to your account dashboard. There, you should see a code snippet that looks like this haiku install .... Make note of that snippet, as we'll be using it below.

Getting Your Animation into Code

With your project saved, we can now pull your animation into a real codebase. To illustrate, we'll first create a minimal website using the React framework, and then embed your animation in it.

For simplicity, these instructions will make use of the popular command-line tool create-react-app. We've also put a sample project on GitHub that you can clone and modify.

  1. Install create-react-app: This popular command-line tool is the easiest way to create a basic React project, which we'll use to demonstrate Haiku's workflow for for engineers. To install it, open Terminal and enter this command (it should take a minute or so to install): npm install -g create-react-app
  2. Create a React project: Once the installation of create-react-app is complete, create an empty folder on your Mac that you can use for code projects. Navigate to that folder using your command line. In that folder, enter the following command: create-react-app my-app
  3. Start the React project: A new folder, my-app, should have been created containing the contents of your new project. Navigate into that my-app subfolder with
    cd my-app. Then start the local development server with npm start. After a short time, a browser window should open automatically, and a simple React website should be displayed.
  4. Install your Haiku animation: With Haiku's developer tools, you can pull your Haiku animation code into any npm-based project with one command. And that's exactly what we'll do inside your new React project: Remember that code snippet from step #6 above? Run it: haiku install ProjectName
  5. Import and embed your Haiku animation: Now that the Haiku animation is available in this project, we can insert it into the code by adding just a couple of lines. Open the file src/App.js in your favorite code editor. At the top of the file, insert import HaikuAnimation from '@haiku/{yourteam-projectname}/react';. Then add the animation somewhere inside your component's render method: <HaikuAnimation />
  6. Save, and watch your animation come to life: Save the code file and your browser should automatically refresh. You should see your animation displayed in the website. (If you had trouble with the coding step, compare your code to our sample project.)
